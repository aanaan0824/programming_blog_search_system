Unity Remote是Unity公司提供的一个移动端同步调试工具，在Unity编辑器中以播放模式运行项目时，该应用程序将与Unity连接。 编辑器的可视输出被发送到设备的屏幕，实时输入被发送回Unity中正在运行的项目。这样可以节省项目调试的时间。

Unity Remote5下载链接 网盘 提取码：1100，也可以自行下载。

要求

  • android的需要有本地Android SDK。
  • 用USB数据线将电脑和android设备连接。
  • android需要打开开发者模式，开启调试模式。
  • 安装 Unity Remote5的App。
  • 在Unity编辑器编辑。

操作步骤

1.[Edit] -> [Project settings] -> [Editor] -> [Unity Remote] -> [Device] 设置为 Any Android

补充

2.打开你的项目，项目的 File-Build Settings 要选成安卓的。用 usb 数据线将电脑连上手机，并打开手机的调试模式。

3.在android设备中打开 Unity Remote5。

4.运行项目 play。
