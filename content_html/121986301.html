一、背景需求

我玩得好的朋友刚创业，搞电商最一块，想做个全套的电商框架系统，希望支持公众号、小程序、H5、pc后台管理等功能，创业初期资金非常紧张，请开发人员做成本太高，然后就咨询我，有没有最低代价就能获取到全套的电商框架系统，我特意在网上找了非常多优秀电商系统对比分析，比如商淘软件、荧火小程序商城、来客推、niushop、shopxo、dsshop、最后发现CRMEB商城最适合我朋友初创公司了，性价比非常高，这么好的开源商城必须分享给大家。

二、CRMEB商城初步了解

1、功能

crmeb是一个全套的电商框架，公众号、小程序、H5、pc后台管理，一套系统，多端适用，提供拼团，秒杀，分销，优惠券等多样化营销活动，支持裂变式分销及推广，部署简单方便，功能强大，包含CRMEB单商户系统、CRMEB Pro系统、CRMEB 多商户系统、PHP版和Java版都有开源。

三、各个商城价钱对比分析

为什么要先分析价钱了，因为这个是我们最关心的，你做得最好，太贵了我也不会用，一定要在价钱和功能上做个权衡对比。

  • tpshop 增强版本(h5+小程序)需要42800元
  • 商淘软件 单商户商城，后台、小程序和H5版本源码价格是 15000元，带上双APP 是25000元
  • 荧火小程序商城 3000元
  • 来客推 小程序+h5+app，23000元
  • niusho 企业版本：4980元 旗舰版本：7890元
  • shopx 不限域名：6800元/12800元/38800元

而 CRMEB开源系统 ：商业版：568元 安装版：2098元 ，性价比最高的就是 CRMEB了，而且看到官网双十二还有非常多优惠活动，这样的价钱和优惠活动是其它电商不具备的。

四、各个商城部署和开发对比分析

如果商城部署和开发太复杂了，人力成本和开发成本和维护成本太大，所以一定要操作简单。

  • dsshop 接口文档基本无、移动版本ui简陋，搞起来肯定有点痛苦，暂时放弃。
  • shopxo小程序UI简陋，演示版本功能太弱，功能不是太强大，暂时放弃。
  • 商淘软件、荧火小程序商城、来客推、niushop、基本只支持php单项语言，不是很好扩展。
  • 来客推、商淘软件 教程太少了，部署比较复杂，暂时放弃。

CRMEB开源系统 接口文档非常详细、有系统功能介绍、系统脑图、目录结构、数据字典、安装教程、二开教程、部署教程、非常简单，部署简单，有自己的论坛，我自己简答部署下页面就刷刷的出来了，感觉非常爽。

七、各个商城技术亮点对比分析

CRMEB商城

java开源优势

  • 有详细的代码注释，有完整系统手册。
  • 基于 SpringBoot 框架开发业界主流。
  • 【前端】Web PC 管理端 vue + element UI。
  • 【前端】移动端使用 UniApp 框架，前后端分离开发。
  • 标准RESTful 接口、标准数据传输，逻辑层次更明确，更多的提高api复用。
  • 支持Redis队列，降低流量高峰，解除耦合，高可用。
  • 无缝事件机制，行为扩展更方便，方便二次开发。
  • 数据导出，方便个性化分析。
  • 数据统计分析,使用ECharts图表统计，实现用户、产品、订单、资金等统计分析。
  • 权限管理，后台多种角色，多重身份权限管理，权限可以控制到按钮级别的操作。
  • Vue表单生成控件，拖拽配置表单，减少前端重复表单工作量，提高前端开发效率。

SpringBoot + Maven + Swagger + Mybatis Plus + Redis + Uniapp +Vue+elementUI

都是市场非常主流和经典的技术，无论是商用还是程序员学习源码，都有极大好处。

php开源优势

  • 自主研发独立客服系统；
  • 管理端页面使用iviewUI开发；
  • PHPExcel数据导出,导出表格更加美观,可视；
  • EasyWeChat部署微信开发,微信接入更加快捷,简单；
  • 使用强大的workerman实现定时任务和长连接；
  • 内置强大灵活的权限管理，可以控制到每一个菜单；
  • 内置组合数据,系统配置,管理碎片化数据；
  • 客户端完善的交互效果和动画；
  • 对接公众号和小程序,并且数据同步；
  • 后台应用form-builder PHP快速生成现代化表单；
  • 高频数据缓存支持高并发；
  • 数据备份和恢复让数据更安全；
  • Restful标准接口、前后端分离，内部预留钩子,二次开发更方便;
  • 长连接减少CPU及内存使用及网络堵塞，减少请求响应时长；
  • 支持队列降低流量高峰，解除耦合，高可用;
  • 一键安装自动检查系统环境一键安装；
  • 移动端使用uni-app进行开发，方便编译成各种格式的代码；

star数目是7.7K，Fork数目是2.5K，基本上在开源项目里面算非常顶尖水平了。超级有实力。

商淘软件、荧火小程序商城、来客推、niushop、shopxo、dsshop基本只支持php单项语言，对java服务端开发者不能快速上手，开发语言兼容适配比较弱，CRMEB有绝对的优势，支持java语言和php语言，基本上满足市场服务端开发。
