今天安装wordpress，按照教程搭好了lamp环境，主机访问IP地址后提示：您的PHP似乎没有安装

运行WordPress所必需的MySQL扩展，在网上找了解决办法全是一些复制粘贴的离谱的办法。

这里一般分两种情况

1：php版本问题

wordpress官方推荐php版本是7.4以上

查看php版本

php -v

这样就可以查看php版本了

如果低于7.4请更换php版本

这个网上有很多教程，这里就不多阐述了

完成后记得重启电脑

2：php的php-mysql扩展

1:ubuntu

安装php-mysql

sudo apt install php-mysql

2:centos

安装php-mysqlnd

yum install php-mysqlnd

重启电脑

至此问题解决
