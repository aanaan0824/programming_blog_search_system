IDE简介

"IDE"是集成开发环境的简称，一般包括编辑器，编译器，调试器等。 而一个好用的IDE不仅能提升代码质量，还能使管理开发工作更简单。

IDE推荐

事先声明：推荐顺序不分前后，只是顺序罢了。

1.Visual Studio

Visual Studio(简称VS)是美国微软公司的一个功能性极强的工具集，并且支持C++，C，Python，HTML等前端后端语言，是一个开发利器，目前最新版本为VS2022 。

VS2022官网 VS2019官网(英文)

其实对于小白我并不是十分推荐这个IDE，因为VS功能太全，新手一时间适应不过来，其次它太"重"了，对电脑内存是一种挑战，动不动就是10多，20多GB。

2.Visual Studio Code

VS的兄弟，最突出的优点是内存占用量低，亲民的界面，并且可以通过各种各样的扩展插件提升性能，几乎支持所有语言，在这里小编强力推荐！

VScode官网

但唯一的缺点是VScode本身不支持C++代码的编译，需要用户自己配置编译器MinGW，这里推荐bat67大大的教程：

整理：Visual Studio Code (vscode) 配置C、C++环境/编写运行C、C++（主要Windows、简要Linux）

3.CLion

这里不得不提一下CLion的制作团队——jetbrains，jetbrains是一个IDE界的隐藏大佬，Python中最有名的Pycharm就是它旗下的产品，好奇的可以去它的官网看看。

好回到正题，CLion最引人注目的就是它的实时代码分析引擎和美观又不失功能性的界面，同样可以安装各种各样的插件提升效率，也是一个不可多得的精品。

但是CLion是付费的，只有30天试用期，至于口袋紧的，还是破解版吧。
