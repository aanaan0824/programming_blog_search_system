二、JDK的安装

1.双击安装包，进入安装界面，直接点击下一步

2.选择开发工具（Development Tools）和指定安装位置，点击下一步

3.完成安装

三、配置环境变量

1找到JDK安装的目录的bin目录，在bin录下Shift+右键（或者Win+R，输入cmd进入bin目录），进入命令行界面进行版本使用查看 java - version 版本信息

由于java这个命令只能在bin目录下才能执行，我们的java程序却不能每次也都一起放在bin目录下，更多的是希望不管java程序放在电脑上的哪一个位置，java这个命令都能起到它的作用。于是我们需要配置我们的环境变量让java这个命令在任何地方都能起到作用。

2.右键计算机属性，进入高级系统设置，再进入环境变量

3.添加系统变量

4.在任意界面Win+R，cmd进入命令行界面，输入 java -version 和 javac ，出现提示信息则配置成功
