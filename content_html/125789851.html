本篇教程只针对小白！因为自己安装的时候踩了很多坑，所以写下来给大家做参考。本文参考Tensorflow安装教程 傻瓜式一键安装

一、Anaconda安装

如果你没有安装过python，安装Anaconda时默认会安装python，非常方便。

下面是具体的安装步骤：

1、下载安装Anaconda

这里安装的版本是Anaconda3-2022.05-Windows-x86_64，官网下载后打开安装，这里建议点Just Me，原因在下一步。

下一步，这里的两个选项都要勾上。第一个选项可以直接配置环境变量，如果不选还需要自己手动配置。我开始选All Users这里第一个选项是灰色的无法勾选，查了很久才看到有人说选Just Me就不会出问题。

直接点安装，一路点Next即可。

2. 验证安装是否成功

打开cmd，输入第一句查看conda版本，输入第二句查看Python版本：

conda -- version
python --version

看到这里的python版本是3.9.12，后续安装Tensorflow需要与python版本适配。

3、Anaconda配置

打开 Anaconda Prompt 切换到国内的镜像源：

  • 中科大镜像源
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/main/

如果切换镜像后当出现下载不了的情况，就先切换默认源，然后再修改另一个可以使用的conda源。

  • 切回默认源
conda config --remove-key channels

二、Tensorflow安装

python3.9对应的是tensorflow2.6.0版本，打开cmd运行，首先创造tensorflow 2.6.0需要的环境：

conda create -n tensorflow pip python=3.9

激活TensorFlow环境：

activate tensorflow

用conda安装tensorflow：

CPU版输入 conda install tensorflow=2.6.0 
GPU版输入 conda install tensorflow-gpu=2.6.0
