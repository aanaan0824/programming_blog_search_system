HTML网址跳转，就是通过一个网址跳转到另一个网址中去，比如你需要在利用输入A网址的情况下，打开的是B网址的网站，那么A网址到B网址之间的跳转就是网址跳转。（此方法也常用于网站的404页面）

在HTML页中，可以使用meta标签进入页面的跳转，此方法可以控制跳转的时间，以及自由化的定义跳转的网址

html跳转代码如下

  1. <meta http-equiv="refresh" content="5;url=https://tsxiaochen.com/">

meta 标签中有一个content属性，表示打开此页面后，多少秒后开启跳转。还有个URL属性，表示跳转的网址

HTML中A标签跳转

HTML中的A标签也可以看做为页面跳转的一种，只是通过鼠标点击的方式进入跳转

html A标签跳转代码如下

  1. <a href="https://tsxiaochen.com">小陈</a>

javascript代码跳转

在网页利用jjs也可以实现页面的跳转或定时跳转

JS跳转代码如下

  1. <script language="javascript" type="text/javascript">
  2. // 以下方式直接跳转（二选一）
  3. window.location.href = 'https://tsxiaochen.com//';
  4. // 五秒以后再跳转（二选一）
  5. setTimeout("javascript:location.href='https://tsxiaochen.com'", 5000);
  6. </script>
