直接说解决办法：安装 apt install libc-ares2 ，然后再去宝塔安装就OK了。

下面是解决问题时的过程

其实主要就是要把这个 libcares.so.2 想办法安装上，我用的腾讯云 debian10。直接执行：

apt install libcares2

肯定是找不到的：

Reading package lists... Done
Building dependency tree
Reading state information... Done
E: Unable to locate package curl-loader

百度了很久，终于知道要安装 libc-ares2，于是执行

apt install libc-ares2

然后再去宝塔安装PHP就可以了，如果你还是安装不上的话，换成下面这个源再试试：
