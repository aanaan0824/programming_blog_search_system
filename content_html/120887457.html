combox是开发winform常用的组件之一，如何添加他的默认值呢！

方法/步骤

  1. 新建一个windows窗体应用程序，这里项目命名为test01

  2. 在默认打开的Form1中，在左边的工具箱拖拉两个控件，label和combox，如图所示。

  3. 1、点击combox控件，点击小箭头

    2、在弹出的菜单中选择“编辑项”

    3、在弹出的编辑器中输入测试文字：---不限---、语文、数学、英语、地理、历史、体育、生物、化学。

    最后点击确定，完成编辑。

    如图所示：

  4. 点击combox控件，在属性面板中点击“Text”，输入默认文本即可。

  5. 效果如图所示：

  6. 动态绑定数据的combox设置默认值，先清空combox的数据，为了显示不同，特意将“不限”放到最后，然后输入代码：

    //添加数据

    //---不限---

    string[] items = new string[9];

    items[0] = "语文";

    items[1] = "数学";

    items[2] = "英语";

    items[3] = "地理";

    items[4] = "历史";

    items[5] = "体育";

    items[6] = "生物";

    items[7] = "化学";

    items[8] = "---不限---";

    comboBox1.Items.AddRange(items);

  7. 代码设置默认值：

    comboBox1.SelectedItem = items[8];

    或者

    comboBox1.Text = items[8];
