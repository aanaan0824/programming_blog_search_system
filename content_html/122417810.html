PHP开启fileinfo扩展的方法

找到flieinfo的路径

find / -name fileinfo

这里是我的目录 每个人的目录都不一样 需要根据情况来换成自己的目录 下面的步骤也是这样

/www/server/php/74/src/ext/fileinfo

进入目录

cd /www/server/php/74/src/ext/fileinfo

执行phpize命令

这里也是根据自己情况替换掉/www/server/php,总之找到bin/phpize所在路径即可

/www/server/php/74/bin/phpize

Configuring for:

PHP Api Version: 20190902

Zend Module Api No: 20190902

Zend Extension Api No: 320190902

执行configure配置

根据自己情况找到bin/php-config的路径

./configure --with-php-config=/www/server/php/74/bin/php-config

编译安装

make && make install

修改php.ini文件

echo 'extension=fileinfo.so' >/www/server/php/74/etc/php.ini
