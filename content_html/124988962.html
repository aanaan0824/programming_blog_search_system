.Net 程序代码混淆加密工具 ILProtector
ILProtector用特定的加密形式将IL转码成只有它自己识别的代码，市面上的反编译器如.NET Reflector, ILSpy, dotPeekn,de4dot等，均不能反编译。
支持 .NET Framework 2.0, 3.0, 3.5, 4, 4.5, 4.5.1, 4.5.2, 4.6, 4.6.1, 4.6.2, 4.7, 4.7.1, 4.7.2
支持 Windows x32/x64

使用超级简单


下载安装即可， 下载地址：http://vgrsoft.net/Download/ILProtector

打开软件，拖入你想加密的exe或者dll，不限制文件个数，最后点击Process按钮即可，完成加密。
这里会生成两个DLL（protect32.dll、protect64.dll）到文件夹中,至于dll的名字无所谓写什么,后面可以使用工具和你的执行文件合并成一个exe
官网介绍的一些限制条件:

C++/CLI混合编码的程序集、类构造函数、泛型以及__arglist关键字方法 可以考虑编码中避开这些限制。
