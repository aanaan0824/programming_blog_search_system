本节，我主要总结下jdk18的下载和安装，因为我看网上这方面资料很少：
jdk其他版本安装类似：
一.下载JDK
1.首先，我们先去oracle官网下载jdk,这个直接百度就可以，当然，你也可以按照我给的链接地址去下载jdk18;

jdk18官方下载地址


2.下载完成后，直接双击.exe文件，按照默认路径安装，当然




安装完成后，点击关闭按钮即可。

二.配置环境变量
1.打开系统高级设置；
1）win10选择（此电脑->属性->高级系统设置->环境变量）
2）当然，你也可以通过命令行打开系统属性；

sysdm.cpl

然后，选择高级，环境变量
3）win11,在搜索栏搜索系统，打开


2.配置环境变量
新建，系统环境变量：变量名是JAVA_HOME，变量值是jdk的具体安装路径；

新建系统用环境变量：变量名是CLASSPATH，变量值是：

.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

编辑系统变量Path: 在里面添加下面一行值

%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin


然后，依次点击确定即可。

三.测试是否配置成功
Win+R键，打开cmd,输入下面命令

java -version


出现标红的部分，则证明jdk已经安装好了。
你也可以测试下面命令，若都没问题就证明你的jdk是安装成功了。

java


javac
