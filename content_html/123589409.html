1. 下载mingw64 和 安装 VSCode

mingw64：mingw64下载地址

VSCode：VScode下载地址

2. 把mingw64/bin路径配置到环境变量

找到mingw64/bin/ ，复制路径

配置方法：我的电脑右键-->属性-->高级系统设置-->环境变量-->下面系统变量-->找Path-->变量-->新建-->把路径复制上去--确定(如果是一个长串，在最后加一个分号(英文的)，再在后面粘贴路径)

3. 打开VScode，安装插件

  • 第一个：搜chinese

  • 第二个：搜Code Runner

  • 第三个：搜C/C++

  • 第四个：搜C/C++ Compile Run

4. 重启VScode

有重启提示，点确定

5. 创建C/C++文件，编写代码

后缀名为.c或.cpp的文件

#include<iostream>
using namespace std;
int main(){
   cout<<"Hello World"<<endl;
}

6. 右键Run Code就可以运行

会有Hello World输出

7. 配置终端输出

在设置中输入 run code config 找到里面的 run in terminal 打勾即可
