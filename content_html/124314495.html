1.在安装前，需要先关掉linux的防火墙

查看防火墙状态命令：

systemctl status firewalld.service

2.关闭防火墙

systemctl stop firewalld.service

3.再次查看防火墙

4. 切换到root用户并输入相应的密码,不然会提示用户权限不够

su root

5.在/usr /local中创建java目录

mkdir /usr/local/java

6.使用filezilla工具连接linux

主机IP是Linux的IP地址

7.将jdk的包传到linux的root目录下

8. 进入root查看文件

9.安装解压命令

yum install tar


10.解压jdk到/usr/local/java

tar -zxvf jdk-8u171-linux-x64.tar.gz -C /usr/local/java/
tar -xvf jdk-8u131-linux-x64.tar.gz -C /usr/local/java/


11.配置环境变量

vi /etc/profile

12. 按下字母i进入可编辑模式，在文件末尾加上

JAVA_HOME=/usr/local/java/jdk
    PATH=$JAVA_HOME/bin:$PATH
    CLASSPATH=$JAVA_HOME/jre/lib/ext:$JAVA_HOME/lib/tools.jar
    export PATH JAVA_HOME CLASSPATH

按下esc输入 ：wq! 保存并退出

13.让刚才的修改立即生效

source /etc/profile

14.查看jdk版本，如果可以看到，证明安装成功

java -version
