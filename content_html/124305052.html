PHPstorm 本地修改同步服务器SFTP配置说明

在有些时候我们需要直接在服务器上修改我们的代码，但是如果用宝塔的编辑器或者其他方式，格式很难同步，也没有代码函数的提示，所以我们需要在本地修改同步服务器。

  • 第一步 打开PHPstorm，并打开你的项目
  • 第二步 点击顶部tools
    选择Deployment -> Configuration

然后会看到Deployment窗口点击左上角的加号 你可以选择SFTP和FTP 我这里演示SFTP的配置


输入服务器名称 自定义就好 自己的标识

添加好后会看到左侧列表已经有你刚刚添加的了 下面点击图上的按钮 添加服务器链接

还是一样的操作左上角添加 输入服务器账号密码 完成后可以点击 Test Connection测试是否可以正常连接 如果连接没有问题点击OK 返回配置

现在可以选择你的服务器的路径了 点击Root Path选择项目路径 如果你是宝塔服务器可以在 www/wwwroot 下找的你的项目目录 选择点击OK 现在距离配置完成就差一步了

点击 Mappings 后点击 Deployment path 选择部署目录 如果你上一步选择了项目目录 直接再次选择根目录即可

如果添加多个服务器部署的话 可以选中所需的服务器 点击左上角对号设置默认选项

到这里配置已经完成 不过如果我们需要修改自动上传还需要设置 一下 看图点上对勾即可

如果我们需要同步某个目录或者上传到服务器 下载到本地某些文件 直接右键目录或文件 看图选择即可

以上就是phpstorm本地修改同步服务器的配置教程 比较入门写的比较细致。
