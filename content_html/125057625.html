1.首先修改mysql字符集，打开my.ini配置文件(MySQL的配置文件)，加入：character_set_server = utf8

2.用命令行方式修改，进入cmd命令，进入mysql的bin目录下，启动mysql；使用命令：show VARIABLES like 'character%'; 查看当前字符集，发现都是gbk等字符集，使用如下命令改变字符集：

mysql> SET character_set_client = utf8 ;

mysql> SET character_set_connection = utf8 ;

mysql> SET character_set_database = utf8 ;

mysql> SET character_set_results = utf8 ;

mysql> SET character_set_server = utf8 ;

3.修改完成后，重启mysql服务，用命令show VARIABLES like 'character%'; 查看当前字符集，已经生效。

4.如果是在修改之前已经创建的数据库，创建过的数据库的字符集也改掉。在navicat for mysql中右键数据库，选择数据库属性， 修改字符集为utf8

5.对每个表点击右键，选择 设计表 ；在“选项”中，将表的字符集进行修改。

6.将varchar类型 字段的的字符集改掉，进入设计表中，选中要修改字段，改为utf8，也可以使用命令的方式修改，命令如下：ALTER TABLE 表名 CHANGE 列名 列名 VARCHAR( 100 ) CHARACTER SET utf8 COLLATE utf8_general_ci NULL DEFAULT NULL;
