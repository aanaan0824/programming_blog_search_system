乌班图安装nginx和php还是很方便的

试用apt install nginx -y 会直接安装成 1.14.0的版本

试用apt install php-fpm -y 会直接安装成7.2的版本

安装后访问ip 出现nginx页面，项目路径是 /var/www/html/ 可以在路径新建一个txt文件测试能否访问到，确认路径是有误。

然后我新建php文件，访问php文件自动变成下载，搞了我两天 头秃吗，各种搜索，各种改nginx.conf文件没啥用，后来看到一个帖子

vim /etc/nginx/site-avaliabel/default

首先 :set number 获取行号

44行 后面加个 index.php

然后 56 57 60 这三行取消注释

然后 systemcal restart nginx 重启nginx服务，再次去访问 php文件，就会正常出现，而不会下载了，在此记录一下，也希望可以帮到各位 可能因为安装环境不一样，所以 大家多试试
