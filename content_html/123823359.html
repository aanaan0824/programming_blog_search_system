vscode社区很好，可以做几乎任何开发的事儿，不限于编写C、python等语言，甚至可以为Arduino、stm32编译、下载程序。那么想要入门vscode，先用它可以编写C就是一个很好的起点：

首先下载一个vscode 网址：Visual Studio Code - Code Editing. Redefined

接下来：

1. 安装就不解释了。

2. 打开vscode，左边有几个控件，点击靠下方那四个方块块的图标：

3. （可选）需要中文的，在打开的这个界面左上角，搜索chiness，安装扩展。

4. 搜索C/C++，安装扩展。

5. 搜索Code Runner，安装扩展。

6. 将搜索框的内容删除，显示的就是已经安装的扩展。

7. 找到已经安装的Code Runner扩展，鼠标左击会显示它的介绍页，右击 选择扩展设置。往下拉，

（1）找到Code-runner：Ignore Selection，勾选；

（2）找到Code-runner:Run In Terminal,勾选；

（3）找到Code-runner:Save File Before Run勾选。

很简单的，一步一步来就行。（图片上第三个没有勾选，不好意思）

8. 点击这个按钮，这个就是工作区。将一个工作文件夹，包含C语言的文件夹（任何文件夹都可以），从桌面上拖拽到vscode，屏幕左侧就会打开这个文件夹，选择一个文件，中间的工作区就enen。

9. 运行程序，右键任意一个位置，选择 第一个 Run Code，就可以运行了。

10. 运行的结果在，屏幕下方，有一个终端的按钮，这是一个cmd，可以执行正常常见cmd的操作。

//附：测试代码

tips：

1.如果报了一个很奇怪的错误 undefined reference to `WinMain@16，那说明这个C文件没有保存！

2. C文件的命名不能有多余的小数点，比如“demo1.模拟.c”，这样子在vscode里的这个插件中 认为不合法，不能编译，当然其他编译器是可以的。

3. 要在vscode中执行C语言，前提是电脑里面有gcc，（其实没有的话，我也不知道可不可以）gcc是一个通过命令行编译C语言的工具，如果以上的步骤不能成功运行的话，估计是缺少这个。具体如何安装gcc，请自行查阅，安装很简单的。
