numpy学习（续）

NumPy 中包含了一个矩阵库 numpy.matlib，该模块中的函数返回的是一个矩阵，而不是 ndarray 对象。

转置矩阵

import numpy as np

lxw = np.arange(16).reshape(4, 4)
print("原数组为：\n", lxw)

print("转置过的数组为：\n", lxw.T)

matlib.empty()

matlib.empty() 函数返回一个新的矩阵

import numpy.matlib

kk = np.matlib.empty((3, 3))    # 填充为随机数据
print(kk)

numpy.matlib.zeros()

numpy.matlib.zeros() 函数创建一个以 0 填充的矩阵

ll = np.matlib.zeros((3, 3))
print(ll)

numpy.matlib.ones()

numpy.matlib.ones()函数创建一个以 1 填充的矩阵

yy = np.matlib.ones((3, 3))
print(yy)

numpy.matlib.eye()

numpy.matlib.eye() 函数返回一个矩阵，对角线元素为 1，其他位置为零

dd = np.matlib.eye(n=3, M=4, k=0, dtype=float)
print(dd)

numpy.matlib.rand()

numpy.matlib.rand() 函数创建一个给定大小的矩阵，数据是随机填充的

sj = np.matlib.rand((3, 3))
print(sj)

二维矩阵

e = np.matrix('1, 2;3, 4')
print(e)

n维数组

r = np.asarray(e)
print(r)
