首先创建一个普通的Java工程：

右键项目找到project structure(快捷键：Ctrl+Alt+Shift+S)

点击Moduls,点击+号，选择web,点击OK

这样就会在工程里生成web模块里面有个WEB-INF里有我们最重要的web.xml,必须的有这个模块

点击在idea运行处的Edit Configurations

点击+号，找到tomcat server下的local(本地),ps:不要选择TomEE server

点击右侧的Configure,配置Tomcat的本地路径。路径到Tomcat的根目录就可以

之后再点回project structure（↑的步骤有位置，这里不放图片了），点击Artifacts,点击+号，选择 web Application Exploded: From Modules...点击OK

修改output directory路径为你自己安装的tomcat下的webapps，点击OK

点击idea运行处的Edit Configurations，点击Deployment选项卡，点击+号，选择Artifact，自动添加了。

会出现这个：

回到Server选项卡，可以修改端口号以及服务器的两个配置改为update classes and resources。

在web目录下新建HTML文件，点击启动服务器，可以通过（自己修改的地址）http://localhost:8080/javaWeb/index.html地址来访问自己的项目。
