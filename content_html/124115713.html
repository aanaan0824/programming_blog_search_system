写这篇文章记录一下我在宝塔上部署laravel项目遇到过的一些问题，以及最后解决办法。

  1. 打开报403错误

    最大可能是你的网站根目录没有设置为public
    在网站配置设置一下就好
  2. 报404错误

    伪静态，没有配置，选择laravel5模板

    或者输入以下代码保存
location / {  
	try_files $uri $uri/ /index.php$is_args$query_string;  
}
  1. 500错误。上面都是nginx报的错，这个500是laravel自己报的错

    首先看debug打开没有，因为这个500错误是比较笼统的，没有打开调试显示详细信息。config/app.php

    修改成true，根据显示的错误详细信息，进一步解决
  2. No application encryption key has been specified. 我的debug打开后就出现了这样的错误

    可能是.env文件加载出错。最后我找到的方法是打开php配置，禁用函数putenv，删除掉，重启php

    再次访问就就可以打开了
  3. 进入后台管理系统时，报了以下这个错，就是一些需要的php扩展没有安装



    安装好以上扩展，然后再次打开 就可以访问
