📜BC1 实践出真知

🍋题目链接：牛客网BC1

🥝考点：考察输出函数的使用

🌽解题思路

仔细观察本题，我们可以发现本题没有输入，只有输出，因此可以得知本题的考点在于考察输出函数的使用

✅ 解法一.printf函数

·printf 是库函数，起到输出作用，因此需要引入头文件——>stdio.h

·因此需要在代码顶端写入 #include<stdio.h>

#include<stdio.h>
int main() //写主函数
{
    printf("Practice makes perfect!");
    return 0;
}

🍊学习plus：main函数

·main函数是程序的入口，代码是从main函数的一行开始执行的！

·一个工程中，有且仅有一个main函数

✅ 解法二.puts函数

#include<stdio.h>
int main()
{
    puts("Practice makes perfect!");
    return 0;
}

🍊学习plus：puts函数

注意！puts函数自带换行效果！

printf("ABCDEF"\n) == puts("ABCDEF")

实战测试：用puts连续纵向输出a、b、c、d、e、f

📜BC3 有容乃大

🍋题目链接：牛客网BC3

🥝考点：考察sizeof运算符的作用

🌽解题思路

仔细观察本题，我们可以发现本题没有输入，单纯考察输出函数使用；但是值得注意的是，这边输出的是数据类型的大小，而使用sizeof则

✅ 解法一. sizeof 运算符

#include<stdio.h>
int main()
{
    printf("The size of short is %d bytes.\n",sizeof(short));
    printf("The size of int is %d bytes.\n",sizeof(int));
    printf("The size of long is %d bytes.\n",sizeof(long));
    printf("The size of long long is %d bytes.\n",sizeof(long long));
}

·C语言标准规定：sizeof(long long) >= sizeof(long) >= sizeof(int)

🍊学习plus：sizeof运算符

·sizeof 是关键字，是C语言中的一个编译时运算符，用于计算变量或数据类型的字节大小。

·sizeof 计算结果的单位是字节

·使用 sizeof 的语法如下：

sizeof (data type)

其中，data type 是要计算大小的数据类型，包括类、结构、共用体和其他用户自定义数据类型。

🍊学习plus：计算机中单位的扩展

📜BC7 缩短二进制

🍋题目链接：牛客网BC7

🥝考点：考察八进制与十六进制的输出

🌽解题思路

本题同样没有输入，只有输出，只是对于输出的形式有所要求，要求以八进制和十六进制进行输出，因此本题考点便是：考察八进制与十六进制的输出问题

✅ 解法一.人为添加前导显示

这里我们为了满足题目的输出要求，人为地在%o 与 %X 前面添加 0 和 0X，使其满足条件

#include <stdio.h>
int main()
{
    printf("0%o 0X%X\n",1234,1234);
    return 0;
}

✅ 解法二.使用“#”控制前导显示

我们这里只需要在%后添加一个‘#’，就会自动加上前导

🍊学习plus：十六进制的大小写控制问题

·十六进制大写：如果我们想要十六进制大写，就用大写的%X

·十六进制小写：反之，如果我们想要十六进制的小写形式，就是用小写的%x

📜BC19 反向输出一个四位数

🍋题目链接：牛客网BC19

🥝考点：循环♻️+模除

🌽解题思路

对于本题，要求反向输出对应的四位数，那么我们以1234为例，看看如何反向挨个输出：

1.输出4：直接用1234模10，就得到了四位数的最后一位

1234 % 10 = 4;

2.输出3：接下来为了得到第三位数，先将1234除10，保留前三位数，再通过模10，获得三位数中的最后一位，即可

    1234 / 10 = 123;
    123 % 10 = 3;

3.输出2：同理，除10保留前2位数，模10取最后一位数

    123 / 10 = 12;
    12 % 10 = 2;

4.输出1：同上

    12 / 10 =1;
    1 % 10 =1;

我们不难发现，从第二步开始，就发生了循环，因此我们采用循环的方式来实现

✅ 解法一.循环模除法

#include <stdio.h>
int main()
{
    int n = 0;
    scanf("%d",&n);//1234
    //倒序拿出
    while(n)
    {
        printf("%d",n%10);
        n = n / 10;
    }
}

这里我们这里解释一下循环条件的判断：当n=1时，进入循环，1%10得到1，n=1/10得到0，此时循环结束，输出结束
