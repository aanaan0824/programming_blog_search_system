5、检查是否安装成功

win + r 或者 搜索栏 搜运行

输入 cmd 点击确定


输入 javac 是否配置环境变量成功

输入 java -version 查看版本号

出现上面页面就算安装成功了

6、未成功的话

%JAVA_HOME%\bin
%JAVA_HOME%\jre\bin
把刚刚配置的环境变量path中的删了替换成上面的两个值再去试执行命令，记住要重新开一个cmd
