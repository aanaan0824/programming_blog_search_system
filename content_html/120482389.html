Unity在VS2019中没有代码提示的问题

  • 1.检查.Net和Unity for VS是否安装
  • 2.Unity关联VS2019
  • 3.查看结果

使用的版本

  • unity 2021
  • visual studio 2019

1.检查.Net和Unity for VS是否安装

在VS2019中 工具->获取工具和功能



这两个钩子要勾上，如果没有勾上的话，要勾上，并且进行安装

2.Unity关联VS2019

在unity中 Edit->preferences ->External Tools中

External Script Editor 选择VS2019， 并点击Regenerate project files

3.查看结果

再打开c#脚本

可以发现出现了unity字样

也可以找到相关的定义

这样就有了代码补全了。
