最近公司项目需要本地调试前端，前端项目使用当前流行的 Ant Design Pro Vue框架，结果发现本机还没有安装nodejs，现将nodejs安装流程分享下：

1.nodejs下载及安装

官方地址：Node.js

选择左边官网推荐的适合大多数用户的版本，我这里是16.15.0

打开下载的.msi的文件，我重新配置的安装路径是D盘目录下，其他默认下一步即可

完成后打开cmd验证一下

输入

#查看node版本
node -v
#查看npm版本
npm -v

没有问题

2.环境变量及下载镜像配置

2.1.配置环境变量

在nodejs安装路径下新建两个文件夹cache和global，用来存放缓存和全局模块（否则后期配置时会默认安装到C盘用户对应的文件夹下）

然后打开cmd输入

#设置缓存
npm config set cache "D:\Program Files\nodejs\cache"
#设置全局模块
npm config set prefix "D:\Program Files\nodejs\global"

然后新建系统变量 NODE_PATH

变量值填 D:\Program Files\nodejs\node_modules

然后调整用户变量PATH里对应的值

修改对应npm的值为global全局模块的路径 D:\Program Files\nodejs\global,

确定。

2.2 设置下载镜像

官网镜像下载较慢，我们使用国内淘宝镜像

打开cmd，输入

npm config set registry https://registry.npm.taobao.org

然后测试一下

npm config get registry

完成安装！

(可选)安装cnpm

npm install cnpm -g --registry=https://registry.npm.taobao.org
