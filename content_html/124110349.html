wamp是非常优秀的windows下优秀的PHP集成环境部署工具，进日在安装完wamp 3.2.6切换到中文后，软件重启提示以下错误（The configuration file contains a systax error on line 44:[EParseError] Mismatched or misplaced quoues on parameter "promptCaption"），点击确认后软件自动退出不能启动。

尝试了很多方法最后得到解决，请按如下方法操作：

解决问题前先到下面网址

下载升级补丁

1、wamp3_x86_x64_language.exe

2、wampserver3_x86_x64_update3.2.8.exe

下载后开始如下操作

方法一、

最快的解决方法是先卸载，再重新安装3.2.6，然后到安装update3.2.8和language补丁，完成后再切换到中文，就不会出问题。

方法二、

第一步、打开wamp安装目录下的wampmanager.conf，将第一行

[main]language = "chinese"

修改

[main]

language = "english"

第二步、打开wampmanager.ini将

[Variables]下所有行用“;”注释掉。如下：

再启动wamp，服务可以正常启动，然后退出服务进行安装update3.2.8和language补丁，完成后再切换到中文，就不会出问题。
