ubuntu安装php扩展包的方法：首先打开命令窗口；然后通过“apt-get install”命令安装需要的php扩展库；最后通过“php -m”命令查看安装的扩展包。

ubuntu 安装php 扩展和查看扩展包

利用ubuntu的软件包下载、安装工具：apt-get

输入下面的命令即可安装 php扩展库mcrypt、curl、gd库、mbstring、simplexml、



  1. apt-get install php-mcrypt;
  2. apt-get install php-curl;
  3. apt-get install php-gd;
  4. apt-get install php-mbstring;
  5. apt-get install php-simplexml;




#php 查看 php扩展包


php -m


#然后具体可以通过管道grep进行搜索了


php --ini #查看


php cli #命令行的路径信息
