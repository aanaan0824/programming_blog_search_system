四、conda的使用

一、环境管理

1、创建python虚拟环境。下面的环境名字自定义。

conda create -n 环境名字 python=3.7

2、激活环境

activate 环境名字

3、查看环境列表

conda info --envs

4、切换到默认环境

conda activate

5、删除环境

conda remove -n 环境名字 --all

6、删除环境中的某个包

conda remove --name 环境名字  包名

7、克隆虚拟环境

conda create -n newenv --clone oldenv

8、查看有哪些虚拟环境

conda env list 
 
conda info -e

二、包管理

1、包查询

conda list

2、包下载

conda install 包名
